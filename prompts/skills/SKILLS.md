# Available Agent Skills

This system provides the following skills for agent operations:

## Core Skills

- **spec-driven-development** - Write structured specifications before coding
- **planning-and-task-breakdown** - Break work into small, verifiable tasks
- **incremental-implementation** - Implement features incrementally with tests
- **test-driven-development** - Write failing tests first, then implement
- **debugging-and-error-recovery** - Systematic root-cause debugging
- **code-review-and-quality** - Five-axis code review
- **git-workflow-and-versioning** - Git workflow practices

## Additional Skills

- **frontend-ui-engineering** - Production-quality UI development
- **api-and-interface-design** - Stable API and interface design
- **context-engineering** - Optimized agent context setup
- **source-driven-development** - Documentation-grounded implementation
- **security-and-hardening** - Security vulnerability prevention
- **performance-optimization** - Application performance tuning
- **ci-cd-and-automation** - CI/CD pipeline setup
- **documentation-and-adrs** - Documentation and architectural decisions
- **shipping-and-launch** - Production deployment preparation

## Using Skills

Skills are invoked via the agent-cli with the appropriate stage and context.
