### 1. Fix `agent-cli` not on PATH
**Problem:** `orchestrator.js` calls `spawnSync("agent-cli", ...)` expecting the binary on PATH, but it was only a local script with no `bin` registration.

**Solution:**
- Added `"bin": { "agent-cli": "./agent-cli/agent-cli.js" }` to `package.json`
- Set a user-local npm prefix (`~/.npm-global`) to avoid needing `sudo`
- Ran `npm link` to register the symlink
