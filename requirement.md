**Build a Minimalist Solar System Web Page**

You are a front-end developer. Create a simple, elegant, and minimalist web page that displays the planets of the solar system using only **HTML, CSS, and vanilla JavaScript** (no frameworks or external libraries).

### Functional Requirements

* Display all 8 planets in order from the Sun:

* Mercury, Venus, Earth, Mars, Jupiter, Saturn, Uranus, Neptune
* Each planet should be represented visually (simple circles or icons are fine).
* Show basic information on interaction (hover or click), including:

  * Planet name
  * Distance from the Sun
  * One key fact

### UI/UX Requirements

* Use a **minimalist design**:

  * Clean layout
  * Neutral or dark background (space theme preferred)
  * Subtle colors for planets
* Planets should be arranged horizontally or in a simple orbital layout.
* Smooth, subtle animations (e.g., hover scaling or gentle orbit movement).
* Responsive design: must work on both desktop and mobile screens.

### Technical Requirements

* Use semantic HTML structure.
* CSS should handle layout, styling, and animations.
* JavaScript should handle interactions (e.g., showing/hiding planet details).
* Keep code modular and readable with comments.

### Optional Enhancements (if time permits)

* Add a simple animation to simulate orbiting.
* Highlight the selected planet.
* Include a toggle for light/dark mode.

### Deliverables

* A single HTML file (or separate HTML, CSS, JS files if preferred).
* Fully working in a browser without dependencies.
